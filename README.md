# Worldwide UK Mortgages

UK mortgage enquiries for expats, seafarers, and residential buyers.

## Stack

- **Astro** for the site framework
- **Vercel** for hosting and deployments
- **Cloudflare** for DNS and edge services
- **GitHub** for source control

## Status

Phase 1 scaffold.

## Local development

```bash
npm install
npm run dev
```

The dev server runs on `http://localhost:4321`.

## Scripts

- `npm run dev` starts the local dev server
- `npm run build` produces a production build in `dist/`
- `npm run preview` serves the production build locally
